# Documentação do Projeto

## Introdução

Esta pasta reúne a documentação de engenharia de software do Sistema de Controle de Estoque, produzida a partir das histórias de usuário e requisitos levantados na disciplina de Engenharia de Software (UNINTER).

Os documentos estão organizados em duas categorias: **requisitos**, que descrevem o que o sistema deve fazer e com que qualidade, e **planejamento**, que acompanha o andamento das tarefas ao longo do desenvolvimento.
